## 方案总结
少前2角色渲染整体采用了基于PBR+NRP结合的方案。
其中，衣服、丝袜等材质使用了结合Ramp的PBR光照算法，在贴图上，对应PBR的渲染部分制作了基础色贴图、法线贴图、RMO贴图和Ramp图。

而脸、头发等材质则使用了NPR的方式来渲染，充分还原二次元卡通渲染的特征。
